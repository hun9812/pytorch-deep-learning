# PyTorch Deep Learning Study

This repository contains my hands-on study of deep learning fundamentals using PyTorch.  
It currently focuses on basic PyTorch concepts, neural network training workflows, and MNIST image classification practice.

## Goals

- Understand core PyTorch concepts such as tensors, autograd, and neural network modules
- Practice building simple neural networks with `torch.nn`
- Learn the basic workflow of training, validation, and model evaluation
- Build a foundation for future projects in computer vision, tabular data, and practical AI applications

## Repository Structure

| Folder | Description |
|---|---|
| `01-pytorch-basics` | Basic PyTorch practice, including tensors, autograd, neural network modules, and simple training workflows |
| `02-mnist-practice` | MNIST classification practice using PyTorch and basic deep learning models |

## Topics Covered

- PyTorch tensors
- Autograd
- `torch.nn` modules
- Loss functions and optimizers
- Training loops
- MNIST image classification
- Basic model evaluation

## Tech Stack

Python, PyTorch, Torchvision, NumPy, Matplotlib

## Status

Ongoing study repository. I plan to expand this repository with CNNs, transfer learning, tabular data modeling, and more practical deep learning projects.
